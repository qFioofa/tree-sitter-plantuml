==================

---
(program (comment (comment_line)))

==================
Single-line comment with no content
==================
'
---
(program (comment (comment_line)))

==================
Single-line comment at the end of line
==================
' Connection comment
---
(program (comment (comment_line)))

==================
Block comment
==================
/' This is a
multi-line
comment '/
---
(program (comment (comment_block)))

==================
Block comment with no content
==================
/' '/
---
(program (comment (comment_block)))

==================
Block comment with special characters
==================
/' Comment with !@#$%^&*()_+ special characters
and numbers 1234567890 '/
---
(program (comment (comment_block)))

==================

==================
/' This is not a real
@startuml

  +method()
}

comment '/
---
(program (comment (comment_block)))

==================
/' This is a block comment

'/
